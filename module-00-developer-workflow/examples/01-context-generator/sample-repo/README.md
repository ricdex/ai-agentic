# E-Commerce Core

Sistema de gestión de pedidos y pagos. Usado como codebase de ejemplo para el módulo 00.

## Dominio

Gestión del ciclo de vida de pedidos (`Order`) desde la creación hasta la entrega,
con procesamiento de pagos (`Payment`) y notificaciones al cliente.

## Stack

- Python 3.11
- pytest para tests

## Correr tests

```bash
pytest tests/ -v
```
